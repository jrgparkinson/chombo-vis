About
This app is a work in progress.
We use vtk.js to render 3D AMR datasets from Chombo.
Several shortcut keys are available:
  • o - [O]pen file
  • a - [A]dd contour
  • c - Toggle [C]ontroller
  • r - [R]eset camera position
For best performance, we recommend using Firefox.
If you have any queries, please go to the GitHub repository.
